# HW01 Алмазов Павел

1. Привести грамматику для языка палиндромов `{ w w^r | w \in {0, 1}* }`. Привести дерево вывода какой-нибудь строки длины не меньше 7 в предложенной грамматике.
#### Решение:

Грамматика:

$$G = \{V_T, V_N, P, S \}$$
$$ V_T = \{0, 1\}$$
$$ V_N = \{S\} $$
$$ P = \{S \rightarrow 0S0 | 1S1 | \epsilon\} $$
